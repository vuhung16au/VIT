# The Patient Journey

A frontend-only Next.js + Tailwind CSS demo that visualizes the holistic healthcare flow from symptom awareness to consultation and recovery actions.

This project follows the FRD in `02.Week03/FRD-Patient-Journey.md` and demonstrates key IxD/UX principles in 4 scenes.


# Screenshots / Demo 

![The Patient Journey Demo](images/patient-journey.gif)


## What This Demo Shows

1. Holistic UX framing with a Journey Map dashboard.
2. Cognitive load reduction through recognition-first specialist cards and progressive disclosure.
3. Social interaction cues via waiting room typing indicators and shared consultation context.
4. Emotional design with empathetic microcopy and trust-building doctor presentation.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS v4
- TypeScript

## Scope

- Frontend only
- Dummy data only
- No backend implementation

## Route Map (4 Screens)

- `/` - Scene 1: Holistic Journey Dashboard
- `/scene-2` - Scene 2: Cognitive Load Reduction
- `/scene-3` - Scene 3: Social Interaction in Telehealth
- `/scene-4` - Scene 4: Emotional Design for Trust and Calm
