# Key Takeaways: UI/UX and IxD

## 1) UX Is a Journey, Not a Button

The first screen intentionally frames the user goal as a complete care path. This prevents feature-level tunnel vision and helps students reason about continuity across screens.

## 2) Reduce Cognitive Load During Illness

- Recognition over recall: icon-driven specialty cards help users avoid medical terminology recall.
- Progressive disclosure: one booking decision at a time prevents overload and decision fatigue.

## 3) Digital Interfaces Can Support Human Connection

- Waiting room typing indicators signal real human attention.
- Shared visual context in consultation supports co-presence and collaborative understanding.

## 4) Emotional Design Is a Core Healthcare Requirement

- Calm palette and whitespace reduce stress.
- Empathetic microcopy turns technical failure into guided recovery.
- Humanized doctor profile improves trust before and during treatment.

## 5) Teaching Discussion Prompts

1. Which UI elements lowered anxiety most effectively?
2. Where does the design prioritize speed versus reassurance?
3. How could accessibility improvements deepen trust further?
4. What might change for older adults or low digital literacy users?
